# Brief v0.2 Implementation Plan — Sync Engine

> **Status:** ✅ SHIPPED (2026-03-29)

**Goal:** Brief generates itself from live sources. Pull from configured sources, render markdown, track source health.

**Tech Stack:** TypeScript, node:child_process (promisified exec), TOML parsing

---

### Task 1: TOML source configuration parser ✅

**Files:** `src/store/config.ts`

Parse `[[sources]]` array from brief.toml with fields: name, type, command, path, target, priority, timeout.

### Task 2: Sync engine core ✅

**Files:** `src/cli/sync.ts`

- Run all sources in parallel via promisified exec
- Per-source timeout with fallback to cached last-good
- Generic JSON-to-markdown renderer
- Source priority ordering (higher priority wins on conflict)
- Render priorities.md, decisions.md, state/*.md from source output
- Write .sources metadata file with health status

### Task 3: Doctor command ✅

**Files:** `src/cli/doctor.ts`

- Show source health (last sync, item count, duration, errors)
- Show config status (sources configured or not)
- Show notification status

### Task 4: Assign command ✅

**Files:** `src/cli/assign.ts`

- `brief assign <person> <item>` creates/updates people/<person>.md
- Sets Current Focus and Assigned sections

### Task 5: Decision command ✅

**Files:** `src/cli/decision.ts`

- `brief decision "description"` appends to decisions.md
- Groups by date, adds timestamp and source attribution

### Task 6: GitHub source adapter ✅

Via generic command source: `gh issue list --json ...` configured in brief.toml.

### Task 7: Scope source adapter ✅

Via generic command source: `scope today --json` configured in brief.toml.
Maps scored items to NOW/TODAY/IGNORED sections.
