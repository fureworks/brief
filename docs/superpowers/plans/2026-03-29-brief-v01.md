# Brief v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the Brief convention + CLI that creates, validates, and reads `.brief/` directories. No sync engine yet — v0.1 validates the convention with manually-written files.

**Architecture:** TypeScript CLI (commander), reads/writes markdown files with YAML frontmatter, hash-based change detection via crypto, integration snippet generation for major AI tools.

**Tech Stack:** TypeScript, commander, js-yaml, node:crypto, vitest
